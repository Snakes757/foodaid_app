Problem it solves: Connects restaurants, bakeries, and supermarkets with NGOs/individuals who can redistribute surplus food to those in need. Reduces waste + fights hunger.

Tech Stack:

Frontend (Web + Mobile):  React Native with Expo

Backend: FastAPI (Python) + Firebase Authentication

Database: Firebase Realtime DB

Geo-location: Google Maps API (to match food donors with NGOs nearby)

Push Notifications: Firebase Cloud Messaging

Payments (optional donations): Stripe / Flutterwave (for Africa)

Hosting: AWS Elastic Beanstalk or Google Cloud
